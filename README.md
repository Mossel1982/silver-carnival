Build a modern, clean web-based accounting and bookkeeping SaaS for freelancers and small businesses.

Core features:
- User authentication (email + Google login)
- Dashboard with overview: total income, expenses, profit, cash flow chart
- Double-entry bookkeeping: add/edit transactions with categories, dates, descriptions, attachments
- Bank reconciliation simulation (upload CSV or manual matching)
- Invoice creation & sending (PDF generation, status tracking)
- Expense tracking with receipt upload (basic OCR description if possible)
- Profit & Loss statement, Balance Sheet, and basic tax reports (export to PDF/CSV)
- Multi-currency support (ZAR + USD as default for South Africa)
- Dark/light mode, fully responsive

Tech stack: Use Next.js or the best modern full-stack setup with Convex (auth + DB) as mentioned on cto.new. Make it beautiful with Tailwind + shadcn/ui.

Start with planning the data models (transactions, accounts, invoices, users) then implement step by step. Make it production-ready with good error handling and data validation.
